# WanderWiz

Welcome to Wanderwiz, a web application designed to generate personalized travel itineraries based on your preferences. Whether you're planning a dream vacation or a quick getaway, Wanderwiz combines your interests, travel experiences, and dietary restrictions to craft a unique and detailed itinerary just for you.

## Features

* **Personalized Itineraries** : Uses Google's Generative AI (Gemini API) to generate travel plans tailored to your preferences.
* **Dynamic Frontend** : Built with HTML, CSS, and JavaScript for a responsive and interactive user experience.
* **Multilingual Support** : Translates the webpage into 50 languages using Google Translate API.
* **Interactive Dropdowns** : Dynamically updates options based on user selections to avoid overwhelming choices.

## Application Architecture

* **Frontend** :
* **HTML** : Structures the user interface.
* **CSS** : Styles the application with modern design techniques and animations.
* **JavaScript** : Manages interactivity, dynamic updates, and AJAX requests.
* **Backend** :
* **Flask** : A lightweight Python web framework for handling API requests and integrating with external services.
* **Google Gemini API** : Powers the NLP engine for generating personalized itineraries.
* **Google Translate API** : Provides multilingual support.
